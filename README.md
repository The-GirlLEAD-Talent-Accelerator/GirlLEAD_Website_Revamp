The GirlLEAD Talent Accelerator is a transformative initiative focused on empowering women in Africa through technology education and skill development. This repository contains the frontend codebase for our official website.

## 🎯 About the Project

GirlLEAD is dedicated to:
- Training and empowering women in technology
- Providing mentorship and networking opportunities
- Supporting career development in tech
- Creating pathways to employment

### Key Statistics
- 1000+ Girls Trained
- 320 Employed Alumni
- 8 Partner Organizations

## 🌟 Features

### 1. Modern UI Components
- **Hero Section**: Dynamic landing page with animated text and impact statistics
- **Testimonials**: Interactive carousel showcasing success stories from alumni
- **Action Cards**: Donation and volunteer recruitment sections
- **Navigation**: Responsive navbar with dark/light mode toggle

### 2. Programs & Initiatives
- TechHER Cohort
- GirlLEAD Bootcamp
- Women in Data Program
- Social Entrepreneurship Training

### 3. Technical Features
- **Responsive Design**: Mobile-first approach for all devices
- **Dark/Light Mode**: Theme toggle with Tailwind CSS
- **Animations**: Smooth transitions with Framer Motion
- **Testimonials**: Touch-friendly carousel with Swiper.js
- **Architecture**: Modular React components for scalability

## 🚀 Getting Started

1. Clone the repository:
```bash
git clone https://github.com/The-GirlLEAD-Talent-Accelerator/GirlLEAD_Frontend.git
cd GirlLEAD_Frontend
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

The development server will be available at `http://localhost:5173` (or next available port).
Production build will be generated in the `dist/` directory.
